# Legal Assistant – Multi-Agent RAG for Indian Legal Documents

A production-style AI-powered Legal Assistant built using **Retrieval-Augmented Generation (RAG)** and a **multi-agent architecture** to provide context-aware answers from Indian legal documents. The application combines hybrid retrieval, reranking, and specialized AI agents to generate reliable, citation-backed legal responses.

## Features

* Multi-agent orchestration for legal reasoning
* Hybrid Retrieval (BM25 + Dense Embeddings)
* BGE Embeddings for semantic search
* BGE Reranker for improved retrieval accuracy
* FastAPI backend
* Streamlit frontend
* Qdrant Cloud vector database
* Groq LLM integration
* Citation-based responses
* Legal section mapping
* Evidence retrieval and validation
* Conversation history
* Analytics and pipeline visualization
* RAGAS evaluation support

## Knowledge Base

The assistant retrieves information from:

* Constitution of India
* Bharatiya Nyaya Sanhita (BNS)
* Bharatiya Nagarik Suraksha Sanhita (BNSS)
* Bharatiya Sakshya Adhiniyam (BSA)

Responses are generated only after retrieving relevant legal context from these indexed documents.

## Multi-Agent Workflow

### Classifier Agent

* Identifies the legal intent
* Determines the offence category
* Routes the query

### Retriever

* Performs hybrid retrieval
* Searches across all legal documents
* Retrieves relevant legal chunks

### Reranker

* Scores retrieved chunks
* Selects the most relevant evidence

### Section Mapper

* Maps applicable legal sections
* Identifies relevant Articles and Acts

### Quote Selector

* Extracts relevant legal text
* Returns supporting quotations

### Validator

* Validates retrieved evidence
* Ensures citations are grounded in source documents

### Remedy Advisor

* Suggests legal remedies
* Provides procedural guidance

### Synthesizer

* Combines outputs from all agents
* Generates the final response

## Installation

Clone the repository:

```bash
git clone https://github.com/Meghanaprakash24/legal-assistant.git

cd legal-assistant
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the environment:

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Environment Variables

Create a `.env` file and add:

```env
GROQ_API_KEY=your_api_key

QDRANT_URL=your_qdrant_url

QDRANT_API_KEY=your_qdrant_api_key
```

Add any additional environment variables required by your project.

## Running the Backend

```bash
cd backend

uvicorn main:app --reload
```

Backend will run at:

```
http://localhost:8000
```

Swagger UI:

```
http://localhost:8000/docs
```

## Running the Frontend

```bash
cd frontend

streamlit run app.py
```
